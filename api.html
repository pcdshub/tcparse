tcparse.parse.get_pou_call_blocks(declaration, implementation)[source]¶

Find all call blocks given a specific POU declaration and implementation. Note that this function is not “smart”. Further calls will be squashed into one. Control flow is not respected.

Given the following declaration:

PROGRAM Main
VAR
        M1: FB_DriveVirtual;
        M1Link: FB_NcAxis;
        bLimitFwdM1 AT %I*: BOOL;
        bLimitBwdM1 AT %I*: BOOL;

END_VAR

and implementation:

M1Link(En := TRUE);
M1(En := TRUE,
   bEnable := TRUE,
   bLimitFwd := bLimitFwdM1,
   bLimitBwd := bLimitBwdM1,
   Axis := M1Link.axis);

M1(En := FALSE);

This function would return the following dictionary:

{'M1': {'En': 'FALSE',
  'bEnable': 'TRUE',
  'bLimitFwd': 'bLimitFwdM1',
  'bLimitBwd': 'bLimitBwdM1',
  'Axis': 'M1Link.axis'},
 'M1Link': {'En': 'TRUE'}
 }
